# Translator

## Leírás
Egy publikus apik használatára épülő fordító és szinoníma kereső kliens alkalmazás.

## Futtatás
Első lépés npm install-val leszedni a szükséges packageket.

Majd ng serve vagy npm start-val elindítani az alkalmazást.

Alkalmazás localhoston fut, publikus apik használatához szükség van internetre.
